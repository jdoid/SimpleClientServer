# SimpleClientServer

This is a very simple Ruby client/server to check up on my server(s) status.  The client is terminal based.  

## Tested Platform

*Ubuntu Server 16.04.3 LTS (xenial)
*Ubuntu Server 14.04.5 LTS (trusty)
